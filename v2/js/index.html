base64原理

base64原理

Base64编码要求把3个8位字节（38=24）转化为4个6位的字节（46=24），之后在6位的前面补两个0，形成8位一个字节的形式。 如果剩下的字符不足3个字节，则用0填充，输出字符使用’=’，因此编码后输出的文本末尾可能会出现1或2个’=’

如何获取base64图片大小

通过base64编码原理我们知道，base64的图片字符流中的每8个字符就有两个是用0补充，而且字符流的末尾还可能存在‘=’号，我们可以通过这个原理计算图片的文件流大小。

getImgByteSize(data) {
		if (data.graphicContents) { // 获取base64图片byte大小
			const equalIndex = data.graphicContents.indexOf('=');  // 获取=号下标
			if (equalIndex > 0) {
				const str = data.graphicContents.substring(0, equalIndex); // 去除=号
				const strLength = str.length;
				const fileLength = strLength - (strLength / 8) * 2; // 真实的图片byte大小
				data.size = Math.floor(fileLength); // 向下取整
			} else {
				const strLength = data.graphicContents.length;
				const fileLength = strLength - (strLength / 8) * 2;
				data.size = Math.floor(fileLength); // 向下取整
			}
		} else {
			data.size = null;
		}
	}

data.graphicContents是后端获取的base64图片的字符串。

前端下载导出Excel

前端通过后端返回的blob格式响应导出下载文件，前端在设置请求响应时设置 responseType: ‘blob’

// res是后端接口返回
 const blob = new Blob([res], { type: 'application/csv' }); // type设置为导出excel
 const fileName = `${new Date().valueOf()}.xls`; // 自定义导出文件命名
 const link = document.createElement('a'); // 创建一个超链接
 link.href = window.URL.createObjectURL(blob); // 创建createObjectURL
 link.download = fileName;
 link.click(); // 触发下载
 window.URL.revokeObjectURL(link.href); // 释放

通过用户在输入框完成操作后多长时间后发请求获取数据，以angular2项目为列

// 关键字搜索trackingList

  searchText$ = new Subject<string>(); // 模糊搜索查询

ngOnInit() {
	this.searchPipeSet();
}
   onSearch(e: string) {
    this.searchText$.next(e);
  }

  searchPipeSet() {
    this.searchText$.asObservable().pipe(
      debounceTime(1000) // 设置时长
    ).subscribe(str => {
      // console.log(str);
	  // 发送请求
      this.pageNation.curentPage = 1;
      const param = {
        keyWords: str,
        pageNum: this.pageNation.curentPage,
        pageSize: this.pageNation.size
      };
      this.reportsService.getAppUserInfoList(param).subscribe(r => {
        console.log(r);
        if (r.msgCode === '200') {
          const resData = r.data.content;
          this.pageNation.totalElements = r.data.totalElements;
          this.listOfData = resData;
        } else {
          this.message.create('error', `${r.msg}`);
        }
      });
    });
  }
